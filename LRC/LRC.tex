    \section{Rendement}

        \paragraph{}
Le rendement de ce code est bon :
\[  Rendement = \frac{n*\text{Taille du message}}{(n+1)*(\text{Taille du message}+1)} \]
Dans le cas que nous étudions (7 messages de 7 bits):
\[  Rendement = \frac{7*7}{(7+1)*(7+1)} \approx 76.6\% \]
